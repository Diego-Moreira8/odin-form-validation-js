O Formulário

Este é um projeto proposto pelo site The Odin Project para prática de formulários gerenciados via JavaScript.
Confira este e outros projetos no meu portfolio!

Precisa ter:

  • Mínimo de 8 caracteres
  • Pelo menos um número
  • Pelo menos uma letra maiúscula
  • Pelo menos uma letra minúscula
  • Pelo menos um caractere especial
  • Não pode ter espaços em branco
